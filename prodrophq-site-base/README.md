
# Prodrophq

Same-day construction material delivery platform for licensed contractors.

This is a Next.js 14 + Tailwind CSS frontend prototype with:

- Home page with marketing copy and "how it works"
- Category browsing for HVAC, plumbing, electrical, carpentry, tools
- Product listing and product detail pages
- Company cart and checkout flow with PO, jobsite, and delivery info
- Simple order tracking page with step-based progress
- Contractor registration page with license fields

## Getting started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the dev server:

   ```bash
   npm run dev
   ```

3. Open http://localhost:3000 in your browser.
